基于hexo的博客搭建指北（持续更新）
发表于 2020-02-13| 更新于 2020-02-14
字数总计:1.7k|阅读时长: 5 分钟

引言

编写网站的方法很多
像HTML，CSS，ASP.net都是可行的
但是相对上述三种，
基于已有的Hexo框架搭建一个网站会更简易，相应的也要求更少的编程知识

博主根据自身的搭建历程，编写了一篇基于Hexo的博客搭建教程
因为博主自身也在尝试的过程中，所以这篇博客会持续更新

环境准备

首先前往Hexo官网安装搭建所必须的环境：


Git和Node.js的安装，大家根据自己的操作系统选择对应的安装包即可
虽然是纯英的，但基本上全部按照default设定来就可以了

准备文件夹&&确认环境

准备好以上环境之后，随便建一个文件夹并命名
右键该文件夹，选择 “Git Bash Here”（不建议使用cmd）
开始进入搭建环节

在开始之前，你可以先确认你的环境已经准备就绪
在Bash窗口中依次输入：

bash

如果显示了对应的版本就说明配置成功
Tips：
如果显示你的电脑缺少npm的话
执行该条指令：

bash

因为我自己在搭建过程中也遇到了npm的一些奇怪的bug
后来查明是因为墙的原因（咳咳）
这条指令是通过淘宝的镜像安装npm
接下来这句话很重要：
如果你执行了该条指令，之后与npm有关的所有指令请切换为cnpm

安装Hexo

在打开的bash窗口中执行指令：

bash

安装结束后，依次输入：

bash

此时进入与“myBlog”同名的文件夹
你会看到如下的文件：

python

如果此时没有报错的话，那么再输入指令：
hexo s
并且在浏览器中输入：
http://localhost:4000
恭喜你，你就可以看到这个demo的预览效果了

预览结束后，关闭网页，返回bash窗口
Ctrl+C关闭服务器

Github准备

首先你需要一个GitHub账号
没有的请先自行注册

之后进入个人主页
GitHub可以使用一个同名仓库的代码托管一个静态站点
所以我们点击左上角的New，如图

进入后看到如下界面：
其中Repository name请填写：
YOURNAME.github.io
比如我就是JOKECHEN66.github.io
之后勾选Public和Initialize this ……两个选项
最后确认创建

随即打开网址https://YOURNAME.github.io/
你就能看到这个仓库托管的静态网页了！

配置SSH密钥

在将你的Hexo demo部署到Git之前，你需要先配置你的SSH密钥
在bash窗口中输入：

bash

可以查看本机的SSH密钥（如果以前生成过的话）
如果没有则开始生成一个SSH密钥：
在bash中依次输入：

bash

用户名就是你的GitHub用户名
邮箱地址同理
注意引号不要遗漏

然后生成SSH密钥

bash

连续回车三次结束该条指令
之后前往：
C:\Users\70928\ .ssh
查看你所生成的密钥，打开文件id_ras.pub
复制其中的内容，并再次前往GitHub

点击右上角的账户，选择Settings一栏，点击进入
并选择SSH and GPG keys一栏

点击右上侧的New SSH key
进入后将你刚刚复制的密钥内容粘贴上去
随便命名一个标签
之后选择创建即可

部署到GitHub

在GitHub上创建完SSH密钥之后
返回已创建的本地文件夹并打开文件 _config.yml
直接滑动到底部

配置如上所示，只需将两处JOKECHEN66更改为你的用户名即可
返回bash窗口
输入指令安装一个部署插件

bash

最后部署上传即可：

bash

等待部署结束以后，打开你的网站，就可以看到预览效果了！

文章编写

如果上述配置均没有问题，那么恭喜你，你已经可以开始编写博客了
在bash窗口中输入：

bash

即可创建一篇新的博客
如果你想增添新的标签
则输入：

bash

即可在source/tags/index.md中找到这个文件
想要创建新的类别亦同理：

bash

至此基础的文章编辑操作已经介绍完毕
如果对写博客有不清楚的地方
建议自行了解.md文件的编辑

设置变更

虽然网站已经被成功部署，但是你会发现文字和设置依旧是默认框架
这时再次打开文件_config.yml

在文件头即可看见一些基础设置，现用表格具体介绍每个参数的作用：

参数名 作用
title 网站主标题
subtitle 网站副标题
description 网站简述
keywords 关键词提示
author 网站作者
language 语言
timezone 时区

其中language的设定里：

参数设定 对应语言
en 英语
zh-CN 简体中文
zh-TW 繁体中文

更多的language和timezone设定可以自行参考Hexo官方文档

这里只对最基础的参数设置做一个简便介绍
其余的功能设置留给给为深入理解
大部分的参数都是有注释标注的，理解起来不困难
如果你已经具备了一定的编程基础，理解和操作起来会更为简单

如果后期更换了主题，那么你需要重新配置各项参数

但有更多基于Hexo实现的其他多样性的主题
